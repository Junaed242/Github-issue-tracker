1. What is the difference between var, let, and const?

   Answer: var is function-scoped, hoisted, and re-declarable. Old way to declare variables.
   Let is block scoped, the value cannot be redeclared in the same scope but can be updated. Introduced in ES6 (modern JavaScript).
   Const is also block scoped but its value can't be redeclared or changed.

3. What is the spread operator (...)?

   Answer: The spread operator expands elements of an array or object into its individual elements or properties.

5. What is the difference between map(), filter(), and forEach()?

   Answer: map() - Creates a new array and Transforms each element
   forEach() - Just runs a function for each element
   filter() - Select elements that pass a specific condition.

7. What is an arrow function?

   Answer: An arrow function is a shorter way to write functions in JavaScript.

9. What are template literals?

   Answer: Template literals allow embedding variables inside strings using backticks `. They use ${} to insert values.
